Module macho

source code

This profile is for the MACH-O file format.

For now this is not a complete implementation - just enough to parse the MACHO core dump style images.

References: https://developer.apple.com/library/mac/#documentation/developertools/conceptual/MachORuntime/Reference/reference.html
