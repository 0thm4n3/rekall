Class Tokenizer

source code

Context-free tokenizer for the efilter language.

This is a very basic pattern-based tokenizer. Any rule from patterns will try to match the next token in the buffer if its state_regex matches the current state. Only meaningful tokens are emitted (not whitespace.)
