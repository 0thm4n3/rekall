 25  """ This module implements a class registry. 
 26   
 27  We scan the memory_plugins directory for all python files and add those classes 
 28  which should be registered into their own lookup tables. These are then ordered 
 29  as required. The rest of Rekall Memory Forensics will then call onto the 
 30  registered classes when needed. 
 31   
 32  The MetaclassRegistry automatically adds any derived class to the base 
 33  class. This means that we do not need to go through a special initializating 
 34  step, as soon as a module is imported, the plugin is registered.
