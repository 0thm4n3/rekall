get_available_addresses(self, start=0)

source code 

Generates address ranges (offset, phys_offset, size) for this AS.

NOTE!! The phys_offset here refers to an address read from the phys_base member of this address space. I.e. it should be possible for callers to call get_available_addresses() and then directly get data via address_space.phys_base.read(phys_offset, length).

Address ranges must be returned ordered.
