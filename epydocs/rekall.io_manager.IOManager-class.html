Create(self, name)

source code  
Creates a new file in the container.

Returns a file like object which should support the context manager
protocol. If the file already exists in the container, overwrite it.

For example:

with self.session.io_manager.Create("foobar") as fd:
   fd.Write("hello world")

Args:
  name: The name of the new file.

This should return a file like object which provides read access to

  IOManagerError: If the file is not found.

GetData(self, name)

source code 

Get the data object stored at container member.

This returns an arbitrary python object which is stored in the named container member. For example, normally a dict or list. This function wraps the Open() method above and add deserialization to retrieve the actual object.

Returns None if the file is not found.

StoreData(self, name, data, **options)

source code 

Stores the data in the named container member.

This serializes the data and stores it in the named member. Not all types of data are serializable, so this may raise. For example, when using JSON to store the data, arbitrary python objects may not be used.
