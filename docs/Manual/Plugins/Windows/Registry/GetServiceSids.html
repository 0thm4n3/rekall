getservicesids

Get the names of services in the Registry and return Calculated SID

Plugin Arguments

hive_offsetsA list of hive offsets as found by hivelist. If not provided we call hivescan ourselves and list the keys on all hives.
hive_regexA regex to filter hive names.If not provided we use all hives.
View Source

The getservicesids command calculates the SIDs for services on a machine. The service names are taken from the registry (“SYSTEM\CurrentControlSet\Services”)

Sample output
