getservicesids

View Source

Get the names of services in the Registry and return Calculated SID

Plugin Arguments

hive-offsets

A list of hive offsets as found by hivelist. If not provided we call hivelist ourselves and list the keys on all hives. (type: ArrayIntParser)

hive_regex

A regex to filter hive names.If not provided we use all hives.

The getservicesids command calculates the SIDs for services on a machine. The service names are taken from the registry (“SYSTEM\CurrentControlSet\Services”)
