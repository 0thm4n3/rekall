vaddump

Dumps out the vad sections to a file

Plugin Arguments

dump_dirPath suitable for dumping files. (Required)
pidOne or more pids of processes to select.
eprocessKernel addresses of eprocess structs.
phys_eprocessPhysical addresses of eprocess structs.
proc_regexA regex to select a process by name.
methodMethod to list processes (Default uses all methods).
View Source

Although you can dump a process executable using the procdump plugin, this only dumps the main executable. For further analysis of a process it is useful to dump its entire address space. Since the address space is discontiguous it is best to dump it out one vad segment at a time.
